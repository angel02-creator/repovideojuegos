# 🎮 Mundo Gamer

Página web sobre videojuegos creada con HTML, CSS y JavaScript.

## Características

- Historia de los videojuegos
- Juegos más famosos
- Diseño moderno
- Juego de preguntas
- Juego de memoria
- Compatible con GitHub Pages

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Cómo usar

1. Descarga el proyecto
2. Ábrelo en Visual Studio Code
3. Ejecuta `index.html`
